If you want to use Realistic Engine Sounds with Randomation Vehicle Physics 2,
import Randomation Vehicle Physics 2, Realistic Engine Sounds and RVP2_RES-Plus.unitypackage.
This unitypackage contain demo scenes, scripts for demo scenes and prefabs for Randomation Vehicle Physics 2.
This prefabs are ready to use with Randomation Vehicle Physics 2.

Get Randomation Vehicle Physics 2 here:
http://randomationmedia.com/rvp2.html
https://github.com/RandomationGames/Randomation-Vehicle-Physics

Read the documentation for tutorials.
